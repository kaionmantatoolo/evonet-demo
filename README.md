# Evonet Drop-in Demo

Local demo app for testing Evonet Drop-in + interaction session creation.

## Setup

1. Copy env template:

```bash
cp .env.example .env.local
```

2. Fill in `.env.local` values.

3. Install and run:

```bash
npm install
npm run dev
```

Open `http://localhost:3000/evonet/dropin-test`.
